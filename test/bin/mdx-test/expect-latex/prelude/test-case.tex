`ocaml-mdx test` can also take a prelude file.

\begin{ocaml}
# Lwt.return 4;;
- : int = 4
\end{ocaml}

It's possible to load a prelude only in a specific environment using
`--prelude=toto:prelude.ml`:

% $MDX env=toto
\begin{ocaml}
# print_endline x;;
42
- : unit = ()
\end{ocaml}

% $MDX version<4.08
\begin{ocaml}
# print_endline x;;
Characters 14-15:
Error: Unbound value x
\end{ocaml}

% $MDX version>=4.08
\begin{ocaml}
# print_endline x;;
Line 1, characters 15-16:
Error: Unbound value x
\end{ocaml}
