#require directives work as expected:

\begin{ocaml}
# #require "lwt";;
# let x = Lwt.return 3;;
val x : int Lwt.t = <abstr>
\end{ocaml}

Top-level `Lwt` values are automatically evaluated with `Lwt_main.run`:

\begin{ocaml}
# Lwt.return 4;;
- : int = 4
# Lwt_list.map_p Lwt.return [1;2;3];;
- : int list = [1; 2; 3]
\end{ocaml}
